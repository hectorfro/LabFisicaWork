\section{Objetivos Generales}
\begin{enumerate}
\item Fomentar el pensamiento computacional para la resolución de problemas científicos mediante la descomposición, el reconocimiento de patrones y la abstracción.
\item Desarrollar competencias en el uso de herramientas computacionales para el análisis de datos experimentales.
\item Enseñar el diseño y análisis de experimentos utilizando metodologías estadísticas y computacionales.
\item Proveer conocimientos fundamentales sobre mediciones, tratamiento de datos y errores en contextos experimentales.
\end{enumerate}

\subsection{Objetivos Específicos}


\begin{enumerate}
\item Introducir a los estudiantes en la computación científica y su relevancia en las ciencias experimentales.
\item  Capacitar a los estudiantes en el uso de software y herramientas computacionales esenciales.
\item  Desarrollar habilidades en diseño experimental y análisis de datos utilizando métodos estadísticos.
\item  Instruir en la redacción de informes científicos y la utilización de procesador de textos \LaTeX.
\item  Proveer conocimientos avanzados en visualización de datos y análisis de regresión.
\end{enumerate}

\subsection{Competencias}

\begin{enumerate}
\item Uso eficiente de herramientas computacionales para la resolución de problemas científicos.
\item Capacidad para diseñar y analizar experimentos de manera efectiva.
\item Habilidad para manejar datos experimentales, analizarlos y presentar resultados de manera clara y precisa.
\item Competencia en la redacción y presentación de informes científicos.
\item Aplicación de métodos estadísticos avanzados para el análisis de datos experimentales.
\item Habilidad para relacionar conceptos teóricos con aplicaciones prácticas.
\end{enumerate}
